# SatisTrain

SatisTrain is a small 3D browser game prototype about growing from hand-dug iron into an automated railway economy.

## Current loop

1. Mine iron ore by hand.
2. Smelt ore into ingots.
3. Craft plates, rods, and rail kits.
4. Lay connected track from the hub to the iron deposit.
5. Build a starter locomotive.
6. Let the train haul ore back to the hub, then upgrade into drills and automatic forging.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
